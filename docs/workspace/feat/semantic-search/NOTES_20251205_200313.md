# NOTES_20251205_200313

## 仕様変更: セマンティック検索ツールの条件付き登録

### 変更内容

`index_status` と `index_refresh` ツールを、セマンティック検索が有効な場合のみ登録するように変更する。

### 理由

- ツールが見えている = セマンティック検索は有効、という前提が成り立つ
- `enabled` フィールドや無効時のエラー返却が不要になる
- ツール一覧がシンプルになる

### 変更対象

- PLAN.md: 新規ツールの仕様を更新
- server.py: 条件付きツール登録、出力の簡素化
- test_server.py: テストの更新

### index_status の出力変更

変更前:
```python
{
    "enabled": True,
    "indexing": False,
    "indexed_count": 660,
    "model": "cl-nagoya/ruri-v3-30m",
    "cache_path": "..."
}
```

変更後:
```python
{
    "indexing": False
}
```
